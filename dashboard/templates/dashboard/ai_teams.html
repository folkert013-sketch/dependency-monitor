AI Teams

Overzicht van alle AI-agents, hun tools en hoe ze samenwerken.

LLM Modellen

Elk team gebruikt twee verschillende Gemini-modellen, afhankelijk van de taak:

Flash Gemini 3 Flash

Snel en kostenefficiënt. Gebruikt voor onderzoek, analyse en schrijftaken waar snelheid belangrijk is.

Pro Gemini 3.1 Pro

Nauwkeurig en grondig. Gebruikt voor synthese, kwaliteitscontrole en eindrapportage waar precisie essentieel is.

Team 1: Dependency Monitor Crew

5 agents — controleert Python dependencies op problemen

1

Dependency Scanner

Flash

Leest requirements.txt en vergelijkt elke package-versie met de nieuwste versie op PyPI. Focust op major updates en security patches.

Requirements Reader PyPI Version Checker
2

API Deprecation Monitor

Flash

Controleert de status van gemonitorde API's (Meta/WhatsApp, Stripe, Django) en signaleert deprecated of bijna-sunset versies.

API Status Checker
3

Security Vulnerability Checker

Flash

Checkt alle dependencies tegen de OSV.dev kwetsbaarheden-database. Focust op CRITICAL en HIGH severity CVE's.

Vulnerability Scanner

Gebruikt resultaten van: Dependency Scanner

4

Breaking Change Analyzer

Flash

Haalt GitHub release notes op voor verouderde packages en identificeert breaking changes en deprecaties.

GitHub Changelog

Gebruikt resultaten van: Dependency Scanner

5

Action Report Writer

Pro

Combineert alle bevindingen tot een rapport met overall status (Kritiek / Waarschuwing / OK), actiestappen, een dev-tip en motiverende quote. Stuurt het rapport per e-mail.

Email Sender

Gebruikt resultaten van: alle vorige agents

Output verwerking

De output van de Action Report Writer wordt geparsed naar bevindingen met severity, categorie en actiestappen. Het rapport wordt opgeslagen in de database en als e-mail verzonden. Geschatte kosten per scan: ~$0.02 - $0.05.

Team 2: Bedrijfsmonitor Crew

4 agents — onderzoekt fiscale ontwikkelingen en schrijft artikelen

1

Fiscaal Onderzoeker

Flash

Doorzoekt Nederlandse overheidsbronnen (belastingdienst.nl, rijksoverheid.nl, kvk.nl) op recente fiscale ontwikkelingen, wetswijzigingen en nieuws voor het MKB.

Gov Site Fetcher Tax News Searcher Deadline Checker
2

Trend Analist

Flash

Analyseert de gevonden ontwikkelingen op urgentie en impact. Selecteert 2-4 onderwerpen en maakt schrijfbriefings met context, kernpunten en doelgroep.

Gebruikt resultaten van: Fiscaal Onderzoeker

3

Blog Schrijver

Flash

Schrijft professionele, goed leesbare artikelen in het Nederlands over fiscale onderwerpen. Elk artikel bevat een pakkende titel, korte intro en uitgebreide body in markdown.

Gebruikt resultaten van: Fiscaal Onderzoeker, Trend Analist

4

Kwaliteitsreviewer

Pro

Controleert artikelen op feitelijke correctheid, bronvermelding en leesbaarheid. Geeft een kwaliteitsscore (1-10) en levert het eindresultaat als gestructureerde JSON.

Gov Site Fetcher

Gebruikt resultaten van: Blog Schrijver

Output verwerking

De JSON output van de Kwaliteitsreviewer wordt geparsed naar individuele artikelen. Elk artikel wordt opgeslagen als concept in de database, klaar voor handmatige review. Via de Django admin kan de status worden gewijzigd naar "Gepubliceerd". Geschatte kosten per onderzoek: ~$0.03 - $0.08.
